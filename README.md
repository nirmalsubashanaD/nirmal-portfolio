# Nirmal Subashana Kalanchige - Software Engineering Portfolio

This is a simple static portfolio website for software engineering internship applications and SEL703 Professional Practice placement applications.

## Files

- `index.html` - main website content
- `style.css` - website styling
- `script.js` - mobile menu and current year script

## How to publish with GitHub Pages

1. Create a new public GitHub repository, for example: `nirmal-portfolio`.
2. Upload `index.html`, `style.css`, `script.js`, and this `README.md` into the repository root folder.
3. Go to the repository on GitHub.
4. Open **Settings**.
5. Go to **Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select branch: `main` and folder: `/root`.
8. Click **Save**.
9. Wait a few minutes. GitHub will publish the site.
10. Your portfolio link will look like: `https://your-github-username.github.io/nirmal-portfolio/`.
